# Privacy Policy - Pixel Perfect Inspector

**Last Updated:** December 19, 2024

## Overview

Pixel Perfect Inspector ("we", "our", or "the extension") is committed to protecting your privacy. This extension works entirely locally in your browser and does not collect, store, or transmit any personal data.

## Data Collection

**We do not collect any data from users.**

The extension operates entirely locally in your browser. No information is sent to external servers, and no data is collected about your browsing activity, personal information, or website content.

## Local Storage

The extension uses Chrome's local storage API (`chrome.storage.local`) to store only the following information locally on your device:

1. **User Preferences:**
   - Selected color theme for the overlay (one of 10 available colors)
   - Tooltip font size preference (between 10px and 20px)

2. **Inspection History:**
   - Last 10 inspected elements (for quick access)
   - Includes only CSS property data from elements you inspected

This data is stored locally on your device and is never transmitted to external servers or third parties.

## Permissions

The extension requires the following permissions:

- **activeTab**: To access the DOM of the active tab when you activate inspection mode
- **scripting**: To inject the inspection script into the active tab when you activate it
- **storage**: To save your preferences locally (color theme and font size)

All permissions are used only for the extension's core functionality of inspecting CSS properties. The extension only accesses the active tab when you explicitly activate inspection mode.

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools. It does not communicate with external servers.

## Data Sharing

**We do not share, sell, or transfer any data to third parties.**

Since we do not collect any data, there is nothing to share.

## Your Rights

Since we do not collect any personal data, there is no personal data to access, modify, or delete. All preferences stored locally can be cleared by uninstalling the extension or clearing Chrome's extension storage.

## Changes to This Policy

If we make any changes to this privacy policy, we will update the "Last Updated" date at the top of this document.

## Contact

If you have any questions about this privacy policy, please contact us through the extension's support channels.

# Política de Privacidade - Pixel Perfect Inspector

**Última Atualização:** 19 de dezembro de 2024

## Visão Geral

O Pixel Perfect Inspector ("nós", "nosso" ou "a extensão") está comprometido em proteger sua privacidade. Esta extensão funciona inteiramente localmente no seu navegador e não coleta, armazena ou transmite nenhum dado pessoal.

## Coleta de Dados

**Não coletamos nenhum dado dos usuários.**

A extensão opera inteiramente localmente no seu navegador. Nenhuma informação é enviada para servidores externos e nenhum dado é coletado sobre sua atividade de navegação, informações pessoais ou conteúdo de sites.

## Armazenamento Local

A extensão usa a API de armazenamento local do Chrome (`chrome.storage.local`) para armazenar apenas as seguintes informações localmente no seu dispositivo:

1. **Preferências do Usuário:**
   - Tema de cor selecionado para o overlay (uma das 10 cores disponíveis)
   - Preferência de tamanho de fonte do tooltip (entre 10px e 20px)

2. **Histórico de Inspeção:**
   - Últimos 10 elementos inspecionados (para acesso rápido)
   - Inclui apenas dados de propriedades CSS dos elementos que você inspecionou

Esses dados são armazenados localmente no seu dispositivo e nunca são transmitidos para servidores externos ou terceiros.

## Permissões

A extensão requer as seguintes permissões:

- **activeTab**: Para acessar o DOM da aba ativa quando você ativa o modo de inspeção
- **scripting**: Para injetar o script de inspeção na aba ativa quando você o ativa
- **storage**: Para salvar suas preferências localmente (tema de cor e tamanho de fonte)

Todas as permissões são usadas apenas para a funcionalidade principal da extensão de inspecionar propriedades CSS. A extensão só acessa a aba ativa quando você ativa explicitamente o modo de inspeção.

## Serviços de Terceiros

Esta extensão não usa serviços de terceiros, análises ou ferramentas de rastreamento. Não se comunica com servidores externos.

## Compartilhamento de Dados

**Não compartilhamos, vendemos ou transferimos nenhum dado para terceiros.**

Como não coletamos nenhum dado, não há nada para compartilhar.

## Seus Direitos

Como não coletamos dados pessoais, não há dados pessoais para acessar, modificar ou excluir. Todas as preferências armazenadas localmente podem ser limpas desinstalando a extensão ou limpando o armazenamento de extensões do Chrome.

## Alterações nesta Política

Se fizermos alterações nesta política de privacidade, atualizaremos a data de "Última Atualização" no topo deste documento.

## Contato

Se você tiver alguma dúvida sobre esta política de privacidade, entre em contato conosco através dos canais de suporte da extensão.
